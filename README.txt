mobile-web-app
基于jquerymobile,spring webmvc实现的移动web开发框架

How to build:
	Clone this Git repository. 
	
	Run 'mvn package'.

  	The resulting TARBALL will under the 'target/' directory.
  	
  	http://localhost:8888/mobile/
  	参照首页提供的静态列表、动态列表、ajax请求演示查看功能
